# **_Screen Cast_**  
A simple alternative to Chromecast which can run on a Raspberry pi
---

#### Created with
* ElectronJS (screen)
* ExpressJS (screen)
* Go-Lang (client)

#### To use

to install dependencies

(You can replace `pnpm` with `npm` or `yarn` if you want to use them instead.)

```sh
$ pnpm install
>
```

To run the screen

```sh
$ pnpm start
>
```

To run the client

```sh
$ cd ./client/ && go run main.go
>
```
